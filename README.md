# ⚽ 2026 W杯 グループステージ突破条件チェッカー

2026 FIFAワールドカップ(カナダ・メキシコ・アメリカ大会)の各出場国について、
その時点でのグループステージ(GS)突破条件を表示する静的ウェブアプリです。

- トップページには **日本代表(グループF)** の突破条件を表示
- 右上の検索窓に国名を入力、または ▼ ボタンのプルダウンから国を選択すると、
  その国の突破条件・順位表・日程に切り替わります(全48カ国、国旗絵文字つき)
- 突破条件は残り試合の全シナリオ(勝/分/負の全組み合わせ)を列挙して自動計算

## 試合結果の更新方法

[`data.js`](data.js) の `MATCHES` 配列で、該当試合の `score` を更新して push するだけです。

```js
// 例: オランダ 1-2 日本 になった場合
{ id: "F1", group: "F", md: 1, date: "2026-06-14", home: "NED", away: "JPN", ..., score: [1, 2] },
```

push すると GitHub Pages が自動で再デプロイされ、全ページの突破条件が再計算されます。

## ローカルでの確認

```sh
python3 -m http.server 8000
# → http://localhost:8000
```

ビルド不要(HTML/CSS/JS のみ)。

## 注意

- 勝ち点が並んだ場合の順位は得失点差・総得点などで決まるため、「確定」判定は
  タイブレークを考慮した保守的な計算(同勝ち点は不利側に倒す)で行っています。
- 各グループ3位は「12グループの3位のうち成績上位8チーム」が突破します。
  3位チームの突破可否は他グループの結果に依存するため、参考表示です。
- 非公式の個人プロジェクトであり、FIFAとは無関係です。
